{%- comment -%} Format an integer with thousands separators (Jekyll's Liquid has no built-in delimiter filter): {% include format-number.html number=12345 %} renders 12,345. Supports values up to 999,999,999. Non-numeric input renders as 0. {%- endcomment -%} {%- assign fn = include.number | plus: 0 -%} {%- if fn >= 1000000 -%}{{ fn | divided_by: 1000000 }},{{ fn | modulo: 1000000 | divided_by: 1000 | prepend: '00' | slice: -3, 3 }},{{ fn | modulo: 1000 | prepend: '00' | slice: -3, 3 }}{%- elsif fn >= 1000 -%}{{ fn | divided_by: 1000 }},{{ fn | modulo: 1000 | prepend: '00' | slice: -3, 3 }}{%- else -%}{{ fn }}{%- endif -%}
